    Color

    Manages Color in Textual.

    All instances where the developer is presented with a color will use this class. The only exception should be when passing things to a Rich renderable, which will need to use the rich_color attribute to perform a conversion.

    lab_to_rgb(lab) ¶

    Convert a CIE-L*ab color to RGB.

    Uses the standard RGB color space with a D65/2 standard illuminant. Conversion passes through the XYZ color space. Cf. http://www.easyrgb.com/en/math.php.

    rgb_to_lab(rgb) ¶

    Convert an RGB color to the CIE-L*ab format.

    Uses the standard RGB color space with a D65/2 standard illuminant. Conversion passes through the XYZ color space. Cf. http://www.easyrgb.com/en/math.php.
