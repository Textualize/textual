# &lt;horizontal&gt;

The `<horizontal>` CSS type represents a position along the horizontal axis.

## Syntax

--8<-- "docs/snippets/type_syntax/horizontal.md"

## Examples

### CSS

```sass
* {
    rule: left;
    rule: center;
    rule: right
}
```

### Python

```py
horizontal = "left"
horizontal = "center"
horizontal = "right"
```
