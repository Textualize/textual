# &lt;vertical&gt;

The `<vertical>` CSS type represents a position along the vertical axis.

## Syntax

--8<-- "docs/snippets/type_syntax/vertical.md"

## Examples

### CSS

```sass
* {
    rule: top;
    rule: middle;
    rule: bottom
}
```

### Python

```py
vertical = "top"
vertical = "middle"
vertical = "bottom"
```
