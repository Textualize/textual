# Markdown

!!! tip "Added in version 0.11.0"

A widget to display a Markdown document.

- [x] Focusable
- [ ] Container


!!! tip

    See [MarkdownViewer](./markdown_viewer.md) for a widget that adds additional features such as a Table of Contents.
